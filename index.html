Meet YEPIE

YEPIE is a powerful, free, and advanced multipurpose Discord bot offering moderation, logging, leveling, welcome messages, and more — just like MEE6, but free forever.

Invite YEPIE

Features

⚙️ Moderation

Ban, kick, mute, and manage users easily with powerful moderation tools.

📊 Leveling System

Rank up users with XP, levels, and custom level-up messages.

📋 Logging

Track bans, joins, deletes, edits, and more with flexible log settings.

👋 Welcome System

Send awesome welcome messages to new members automatically.

🎮 Fun Commands

Jokes, memes, 8ball, games, and much more to engage your community.

💡 Easy to Use

Just invite and run `?help`. No setup required, but tons of customizations!

📘 Commands

🛠️ Moderation

• ban — ?ban [user] [reason]

• unban — ?unban [user] [case]

• kick — ?kick [user] [reason]

• mute — ?mute [user] [reason]

• unmute — ?unmute [user] [case]

• warn — ?warn [user] [reason]

• unwarn — ?unwarn [user] [case]

📈 Leveling

• leveling setup — ?leveling setup

• leveling channel — ?leveling channel [channel]

• leveling message — ?leveling message [message]

📋 Logs

• logs setup — ?logs setup

• logs set — ?logs set [type] [channel]

• logs disable — ?logs disable

👋 Welcome

• welcome setup — ?welcome setup [channel]

• welcome disable — ?welcome disable

📦 General

• role info — ?role info [role]

• role add — ?role add [user] [role]

• role remove — ?role remove [user] [role]

• role list — ?role list

• help — ?help
